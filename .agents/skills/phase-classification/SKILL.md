# Phase Classification

Use this skill when a phase needs a single controlled classification.

## Steps
1. Verify the current checkpoint and scope.
2. Inspect only the files and contracts relevant to the requested phase.
3. Determine the narrowest blocker or readiness state from repo evidence.
4. Choose exactly one allowed classification from the prompt.

## Guardrails
- Do not broaden into roadmap planning unless asked.
- Do not convert partial coverage into “safe to implement”.

## 🔧 استكشاف الأخطاء

### المشكلة: لا تظهر نافذة Login

**السبب**: الملفات غير موجودة في المكان الصحيح

**الحل**:
1. افتح Console (اضغط F12)
2. ستجد أخطاء مثل: `Failed to load js/auth.js`
3. تأكد من أن المجلد اسمه `js` وليس `frontend-js`

### المشكلة: "Failed to fetch" مازال يظهر

**السبب**: لم يتم تفعيل Dev Mode

**الحل**:
1. افتح ملف `js/auth.js`
2. في السطر 7، تأكد من:
   ```javascript
   DEV_MODE: true,  // يجب أن يكون true
   ```

### المشكلة: البيانات كلها أصفار

**السبب**: هذا طبيعي! البيانات تجريبية

**الحل**: لا حل! هذا متوقع في وضع التطوير.
عندما يكون السيرفر جاهز، ستظهر البيانات الحقيقية.
